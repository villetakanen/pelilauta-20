The tray is a multi-mode responsive navigation sidebar. It adapts its layout based on container width:
- **Mobile (< 621px):** Toggleable drawer overlay with scrim.
- **Tablet (>= 621px):** Persistent icon rail that expands into an overlay modal on click.
- **Desktop (>= 780px):** Persistent icon rail that pushes content when expanded (no scrim).

## Navigation Hierarchy

Hierarchical navigation is achieved by pairing `TrayButton` (primary categories) with `TrayLinkGroup` and `TrayLink` (sub-pages).

### Pattern: Category Navigation

In this pattern, a primary `TrayButton` acts as the group header and index link. All sub-pages are nested within a `TrayLinkGroup`.

## Components

### TrayButton
The primary interactive element. In "Rail Mode" (collapsed), it displays only the icon. In "Expanded Mode", it displays both the icon and label.

### TrayLinkGroup
A semantic container (`<ul>`) for sub-navigation links. 
- **Visibility:** Automatically hidden via `@container` queries when the Tray is in Rail Mode.
- **Accessibility:** Uses `aria-label` to provide context for the link group.

### TrayLink
An individual navigation item (`<li><a>`) used within a group.
- **Active State:** Uses `aria-current="page"` for semantic highlighting.
- **Overflow:** Automatically applies ellipsis to long labels.

## Properties

### TrayLinkGroup
| Prop        | Type     | Default                  | Description                               |
| :---------- | :------- | :----------------------- | :---------------------------------------- |
| `ariaLabel` | `string` | "Secondary Navigation"   | Accessible label for the `<ul>` element.    |

### TrayLink
| Prop     | Type      | Default | Description                               |
| :------- | :-------- | :------ | :---------------------------------------- |
| `href`   | `string`  | (Req)   | Navigation destination.                   |
| `label`  | `string`  | (Req)   | Text label for the link.                  |
| `active` | `boolean` | `false` | Whether the link is the current resource. |
