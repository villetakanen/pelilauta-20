# Tray Component

The tray is a multi-mode responsive navigation sidebar. It adapts its layout based on container width:
- **Small (< 621px):** Toggleable drawer overlay with scrim.
- **Medium (>= 621px):** Persistent icon rail that expands into an overlay modal on click.
- **Large (>= 780px):** Persistent icon rail that pushes content when expanded (no scrim).
