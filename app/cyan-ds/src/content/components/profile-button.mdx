import ProfileButton from "@cyan/components/ProfileButton.svelte";

The `ProfileButton` is the single entry point for identity navigation in all Cyan apps. It switches between a login prompt, a loading skeleton, and an authenticated avatar based on session state.

## Overview

It is typically placed in the `AppBar`'s trailing-action slot and reacts to the user's authentication state.

## Props Table

| Prop | Type | Default | Description |
|---|---|---|---|
| `loading` | `boolean` | `false` | Session is rehydrating. |
| `nick` | `string` | `undefined` | Authenticated user's display name. |
| `photoURL` | `string` | `undefined` | Authenticated user's avatar image URL. |
| `loginHref` | `string` | `"/login"` | Anonymous login destination. |
| `settingsHref` | `string` | `"/settings"` | Authenticated settings destination. |
