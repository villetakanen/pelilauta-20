<section class="cn-content-prose">
  `AppFooter` is the shell-level footer primitive rendered by `AppShell` in non-modal layouts when host apps provide footer slots. It is structure-only: hosts own content, the DS owns semantics and layout tokens.

  ## Contract Surface

  - **`footer-body` slot**: primary footer content (columns, link groups, summaries)
  - **`app-footer-credits` slot**: optional attribution/credits line

  If credits are omitted, the component does not render an empty placeholder block.
